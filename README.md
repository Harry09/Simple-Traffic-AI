# Simple-Traffic-AI
This is my simple traffic AI which can drive road and select direction on crossroad. Crossroad don't have traffic light and car don't have collision. (ugh poor english)

# Help
- 1 - Select street
- 2 - Select crossroad
- R - spawn car
- W/S/A/D - move camera
- Mouse scroll - zoom
